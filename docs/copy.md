
> 复制组件 

``` javascript
//使用
<fcw-copy
  value="我是复制的内容"
  icon="gf-fuzhi"
>
<fcw-copy/>
```

<font color=#0077AA>icon</font> | 图标icon | <font color=red>string</font> |  <font color=#669900>gf-fuzhi</font>  | <font color=#669900>gf-fuzhi</font>
